# Análisis de Datos Dataset: SUPERSTORE

## Descripción breve del dataset utilizado.
Este conjunto de datos, que contiene información de diversas regiones y segmentos, permite explorar tendencias, patrones y correlaciones en las ventas y las preferencias de los clientes . El conjunto de datos abarca las transacciones de venta, lo que permite a investigadores y analistas comprender los patrones de compra, identificar productos de alta demanda y evaluar la eficacia de diferentes modalidades de envío.

Además, el conjunto de datos ofrece la oportunidad de examinar el impacto de diversos factores, como descuentos, ubicaciones geográficas y categorías de productos, en la rentabilidad . Al analizar este conjunto de datos, las empresas y los expertos en datos pueden descubrir información valiosa para optimizar las estrategias de precios, la gestión de la cadena de suministro y la interacción con el cliente .

## Objetivo
Aplicar **análisis exploratorio de datos** con Python y sus libreerias (Pandas, Plotly) e identificar puntos a mejora en el SUPERSTORE

## Fuente del dataset y link hacia Kaggle:
https://www.kaggle.com/datasets/ishanshrivastava28/superstore-sales/data?select=Superstore.csv

## Etapas:
- Carga (import pandas)
- Exploración Inicial del Dataset (uso de info(), shape, head(), tail(), revisión de nombres de columnas y tipos de datos.)
- Análisis de calidad de datos (identificación de valores nulos, vacíos, duplicados)
- Estadística Descriptiva (análisis de variables numéricas y categóricas)
- Visualizaciones con Plotly (gráficos adecuados apra el análisis)
- Consultas SQL con DuckDB 
- Hallazgos y conclusiones
